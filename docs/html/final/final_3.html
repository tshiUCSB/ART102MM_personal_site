SONVA A canvas painted with sounds

Welcome to Sonva. This is an art tool that allows you to use sounds as a drawing tool to create art.
Click on this text to learn more about how to use Sonva. You can also click anywhere else at any time to skip the walkthrough.
The pencil tool lets you draw lines with varying thickness and direction depending on the sound.
The pencil tool automatically moves horizontally and lets you draw lines with varying thickness and direction depending on the sound.
It has 3 sliders for fine-tuned control. The first slider controls the speed that it automatically draws horizontally and whether it draws to the right or the left. A larger positive value moves faster to the right and a larger negative value moves faster to the left.
The second slider controls the max thickness of the line and how sensitive thickness is to changes in the amplitude of the sound.
The third slider controls how sensitive the vertical speed of the pencil is to changes in the frequency of the sound.
You can select the microphone button to start listening from your mic. To stop it from listening, click again to deselect. While the mic is listening, you can click anywhere on the canvas to start drawing a line.
